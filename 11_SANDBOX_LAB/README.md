# Sandbox Lab

Safe testing environment for OpenHuman, Hermes, OpenHands, or any future agent framework.

## Rule

No external agent gets direct access to real repos, secrets, production endpoints, or main branch until it passes the promotion ladder.

## Promotion Ladder

L0 Dry Run → L1 Read Only → L2 Simulated Patch → L3 Sandboxed Execute → L4 Draft PR → L5 Trusted Narrow Autonomy
