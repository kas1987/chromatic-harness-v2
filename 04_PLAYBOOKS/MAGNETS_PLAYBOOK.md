# Magnets Playbook

## Purpose

Defines how Magnets are attached, what they collect, and how they report.

## Attachment Rule

Every mission must attach at least:

- Intent Magnet
- Scope Magnet
- Execution Magnet
- Confidence Magnet

High-risk missions must also attach:

- Security Magnet
- Validation Magnet
- Cost Magnet

## Output Rule

Every Magnet emits structured events to `07_LOGS_AND_AUDIT/magnet_events/` and a summary report to the Agent Lead.
