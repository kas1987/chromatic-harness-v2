# Model Routing Playbook

## Routing Defaults

| Task Type | Preferred Model Class |
|---|---|
| strategic planning | Claude or ChatGPT |
| tool-heavy orchestration | ChatGPT |
| repo implementation | Codex or code-specialized agent |
| broad-context comparison | Gemini |
| current web research | browsing-capable model |
| cheap repeated triage | local LLM |
| final audit | Claude + ChatGPT independent passes |
