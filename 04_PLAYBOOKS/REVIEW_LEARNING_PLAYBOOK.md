# Review Learning Playbook

## Purpose

Convert repeated reviewer feedback into durable harness rules, tests, docs, and repo standards.

## Learning Sources

- `07_LOGS_AND_AUDIT/review_intake/findings.jsonl`
- `07_LOGS_AND_AUDIT/review_intake/resolution_log.jsonl`
- recurring failed checks
- repeated reviewer comments
- blocked findings

## Learning Actions

| Pattern | Action |
|---|---|
| repeated lint/style issue | Add formatter/linter rule |
| repeated test failure | Add regression test template |
| repeated architecture concern | Update architecture standards |
| repeated repo hygiene issue | Update ChromaticTrees/repo rules |
| repeated vague task | Improve mission packet template |

## Review Cadence

- Weekly during active PR cycles.
- Immediately after severe collision or failed auto-patch.
