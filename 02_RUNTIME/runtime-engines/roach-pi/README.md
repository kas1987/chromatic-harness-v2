# roach-pi (git submodule)

This directory is populated by:

```bash
git submodule update --init 02_RUNTIME/runtime-engines/roach-pi
```

Or `scripts/init_roach_pi_submodule.ps1` / `scripts/init_roach_pi_submodule.sh`.

Until initialized, Chromatic uses **stub** execution in `02_RUNTIME/adapters/roach-pi-adapter.ts`.

Upstream: https://github.com/tmdgusya/roach-pi
