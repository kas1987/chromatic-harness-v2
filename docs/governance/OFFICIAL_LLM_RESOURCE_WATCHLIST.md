## Review template

For each source, capture:

- Date reviewed (UTC)
- What changed
- Routing impact (none/low/medium/high)
- Governance impact (none/low/medium/high)
- Action required (yes/no)
- Linked bead id if action required

## Cadence

- Minimum every 14 days.
- Mandatory before major routing matrix edits.
- Mandatory before production readiness sign-off.
