# Incident Log

Use this file for high and critical operational events that require human-readable review.

## Incident Template

```md
## INCIDENT-YYYYMMDD-001: [Title]

| Field | Value |
|---|---|
| Severity | High / Critical |
| Status | Open / Active / Resolved / Escalated |
| Event IDs | evt_... |
| Detected At | YYYY-MM-DD HH:MM |
| Source | IDE / terminal / agent / CI |
| Owner | Human / Agent |

### Summary

### Impact

### Evidence

### Immediate Containment

### Root Cause

### Resolution

### Follow-Up Actions

### Learning Added
```
