# Collision Register

Use this file when multiple writers, agents, IDEs, or terminals touch the same file and risk overwriting each other.

## Collision Template

```md
## COLLISION-YYYYMMDD-001: [Affected File]

| Field | Value |
|---|---|
| Status | Open / Resolving / Resolved / Escalated |
| Severity | Medium / High / Critical |
| Event IDs | evt_... |
| Files Affected | path/to/file |
| Resolver | Agent / Human |

### Writers

| Writer | Surface | Session | Claimed Task |
|---|---|---|---|
|  |  |  |  |

### What Happened

### Risk

### Resolution Decision

### Final State

### Prevention Learning
```
