# PDR: Chromatic Frontend Workflow Console

## Purpose

Create a frontend command surface for Chromatic Harness v2.

## Required Views

1. Mission Dashboard
2. Agent Status
3. Magnet Event Stream
4. Confidence/Risk Panel
5. Beads Queue
6. Independent Review Panel
7. PDR Generator
8. Action Launcher
9. Sandbox Lab Results

## Quick Actions

- request independent review
- rerun validation
- create Bead
- dispatch agent
- generate PDR
- pause mission
- quarantine agent
- export audit report

## AION/OpenCloud Evaluation Checklist

Before adopting a repo as the base frontend, verify:

- license compatibility
- Docker support
- frontend stack
- backend/API architecture
- auth model
- extensibility/plugin model
- WebSocket or event-stream capability
- local deployment path
- ease of adding dashboards/actions
